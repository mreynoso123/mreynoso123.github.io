HTML provides the basic structure of a webpage by defining elements and how content is organized.
Semantic HTML uses meaningful tags to clearly describe the purpose of different parts of a page.
These tags help browsers understand how content is related and displayed. Semantic structure also improves accessibility for assistive technologies and makes webpages easier to interpret by search engines.
As an example, this webpage uses the following HTML tags:
  • <header>
  • <nav>
  • <main>
  • <section>
